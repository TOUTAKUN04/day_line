## App Name
Day Line

## Short Description
Daily planner with reminders, alarms, and walk intensity tracking.

## Full Description
Day Line helps you plan tasks, track activity, and stay consistent every day.

Key features:
- Task planning with date/time, categories, tags, and notes
- Recurring tasks (daily, weekly, monthly)
- Reminder and start-time alarm support
- Home screen widgets for daily tasks and Walk-o-meter
- Walk-o-meter tracking with intensity score and weekly progress
- Weather summary based on device location

What is new in v1.1.2:
- Weekly Intensity Score now follows a fixed Monday-Sunday calendar week
- Weekly score line resets at local Monday 12:00 AM
- Stability and release packaging updates
